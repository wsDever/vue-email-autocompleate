## vue图片展示插件

#### 主要功能
###### 图片查看，支持图片放大、缩小、翻转等功能

#### 使用方法
###### 安装
> npm i vue-email-autocompleate

###### 引入
```
import emailAutocompleate from vue-email-autocompleate

```

###### 页面中添加组件
```
<email-input></email-input>
```

###### 使用
```
Vue.use(emailAutocompleate)
```
#### 参数说明
```
className: String         ---- 自定义样式
placeHolder: String       ---- 提示
suffixs: Array            ---- 邮箱后缀
```
